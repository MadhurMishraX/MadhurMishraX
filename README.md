### about

I build tools around the problems I keep running into: studying, tracking progress, handling messy information, and connecting software to real hardware.

Right now my work sits around three areas:

```txt
education systems   -> dashboards, assignments, analytics, LaTeX notes
health hardware     -> kiosk flows, RFID/QR access, Arduino control
personal tools      -> time tracking, study workflows, productivity experiments
```

I am still early. That is the fun part. The projects are not museum pieces; they are things I keep breaking, fixing, and improving.
